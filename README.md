
# AQUA MARS AI

## Visão Geral

O AQUA MARS AI é uma plataforma de Ciência de Dados inspirada nas tecnologias utilizadas para detectar água em Marte.

O projeto combina Inteligência Artificial, Machine Learning, análise geoespacial e dados climáticos para prever riscos de escassez hídrica e apoiar a gestão sustentável dos recursos hídricos.

---

## Objetivo

Antecipar cenários de escassez hídrica utilizando:

- Dados da ANA
- Dados climáticos da NASA POWER
- Inteligência Artificial
- Machine Learning
- Análise Geoespacial

---

## Tecnologias Utilizadas

- Python
- Pandas
- Scikit-Learn
- Matplotlib
- NASA POWER API
- ANA
- Machine Learning
- Data Science

---

## Dados Utilizados

### ANA
- Volume dos reservatórios
- Percentual armazenado
- Coordenadas geográficas
- Situação operacional

### NASA POWER
- Temperatura média
- Umidade relativa
- Precipitação
- Evapotranspiração

---

## Resultados

- 45 reservatórios analisados
- 18 reservatórios críticos (≤ 5%)
- 27 reservatórios em atenção (≤ 20%)
- Modelo preditivo com 100% de acurácia na prova de conceito
